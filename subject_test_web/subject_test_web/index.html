Test Web Front-End

Sujet : Le candidat devra à partir d'un object .JSON fourni, générer une bibliothèque d'images avec comme propriétés principales :

    1) Le rendu d'une top bar avec un bouton permettant la recherche d'image (la fonctionnalité de recherche est non prioritaire et facultative).
    2) Le rendu des images de manière homogène avec leur titre contenu dans l'object.
    3) L'ajout d'un overlay scrollable sur l'image avec du texte contenu dans l'object au focus de l'image.
    4) L'affichage d'un encart de couleur différençiant les notes en 3 catégories (mauvaise / moyenne / bonne). La note de l'image étant contenue dans l'object.

Contraintes :

    1) Le candidat devra écrire le code javascript / html / css sans utiliser de ressources chargées à l'extérieur du projet, impliquant de n'utiliser aucune librairie externe ni framework.
    2) Le candidat devra utiliser comme fonts celles founies dans les ressources du projet et comme images celles fournies dans les ressources du projet.
